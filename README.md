# novendemosgalletas
Sitio estático para GitHub Pages del Grupo Scout Santa Isabel 488.

## Desarrollo local
- Abre `index.html` en tu navegador o usa un servidor estático (`python -m http.server`).

## Publicar en GitHub Pages
- Crea el repositorio en GitHub con estos archivos.
- Habilita Pages desde `Settings` → `Pages` apuntando a la rama `main` (carpeta `/` raíz).
- La página quedará disponible en la URL que GitHub asigne al proyecto o al dominio personalizado que configures.
